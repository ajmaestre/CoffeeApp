Preparación de los datos

  • En esta etapa de la metodología prepararemos los datos para adecuarlos a las técnicas de minería de datos que se van a emplear sobre ellos. Seleccionaremos aquellos datos que vamos a utilizar para la construcción del modelo, realizaremos su respectiva limpieza, añadiremos nuevos datos de ser necesario y les daremos el formato adecuado para la fase de modelado.
